the following command is to display the history of the last 2 commits:
git log --pretty='%H' -2

"git log": to display the history
"--pretty": Show commits in an alternate format.
"%H": Commit hash
"-2": Show only the last 2 commits(it depends on how much u need).
 
 For more informations visit: https://git-scm.com/book/en/v2/Git-Basics-Viewing-the-Commit-History
